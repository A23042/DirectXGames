作品概要
■制作中ゲーム
  授業で制作中の作品です。1人で開発しています。
　私が書いたコードは「S.Matsunaga」で検索すると出てきます。
　実行してEキーを押すとMapEditor、Pキーでプレイシーンへ移行します。
　プレイシーンではボールと直方体を設置して衝突判定のテストをしています。
　
操作方法
　■MapEditor
　・基本操作
	右クリック押しながらマウス操作で視点移動
	右クリック押しながらWASDで移動、EQで上下移動
	左上「NEWOBJ」でオブジェクトの生成、「OBJINFO」で選択されているオブジェクト移動、回転、スケールなどの調整
	「MENW」で選択したステージ番号のcsv読み込み、書き込み
	右上「SETTING」でカメラ移動速度、視点感度の変更
	マウス左クリックでオブジェクト選択
　・ブジェクト選択状態
	Nキーで選択解除
	Ctlr+Dで複製
	DELETEで削除
	Rで移動、回転、スケールのモード変更
	モードでギズモをドラッグで対応するオブジェクトの操作

　■PlayeScene
　・基本操作
	WASD：左画面前後移動回転、LCtlr：前方に加速
	矢印キー：右画面前後移動回転、RCtlr：前方に加速
